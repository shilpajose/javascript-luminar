Dom methods
-----------

dom is an object(document object)

- document.getElementsByTagName('tagname)
- document.getElementsByClassName('classname')
- document.getelementById('id')
Instead of these we can use :-
- document.querySelector('tagname/.class/#id')  // only one output
- document.querySelectorAll('tagname/.class/#id') // all
